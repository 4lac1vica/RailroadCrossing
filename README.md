# RC

Components : 6 LEDS (4 reds and 2 whites), one potentiometer and 2 servos(at 17.01.2026 update)


## RO 
Acest proiect propune proiectarea unui prototip de bariera de tren, folosindu-ma de un potentiometru, LED-uri si 2 servomotoare. Acestea au fost lipite pe un placaj din poliuretan. 
Link (pentru a putea fi observata evolutia montajului): https://www.youtube.com/watch?v=bM316-picHM&list=PLA1ZSYjgQUrnU9FbLbrIvAognBKYMzcuv

## EN 

This project proposes the design of a train barrier prototype, using a potentiometer, LEDs, and 2 servomotors. These were attached to a polyurethane board.
Link : https://www.youtube.com/watch?v=bM316-picHM&list=PLA1ZSYjgQUrnU9FbLbrIvAognBKYMzcuv
